# Eventure - Social Event Management App

A full-stack Android application that enables users to create, discover, and manage events in real time using Firebase. Developed as part of CMPUT 301, this project focuses on scalable backend integration, collaborative development practices, and user-centered mobile design.

## Overview

Eventure is a mobile event management platform designed to streamline event organization and participation. The application allows users to create events, register as attendees, manage event capacity, and view updates dynamically through cloud-based data synchronization.

The system integrates Firebase Authentication and Firestore to provide secure user management and real-time database functionality, enabling seamless multi-user interaction.

## Features

### User Management
* User registration and authentication via Firebase
* Profile creation and editing
* Secure user-event association

### Event Management
* Create and edit events
* Set and enforce event capacity limits
* View and manage attendee lists
* Join or withdraw from events
* Real-time event updates via Firestore

### Cloud Integration
* Firebase Firestore real-time database
* Persistent cloud-based storage
* Scalable user-event data modeling
